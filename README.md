## Bobby
---
##### Bobby is a package to get prime factor of numbers currently supported for (*nix)
---
In order to use it:

1. `pip install bobby`
2. In terminal
    ```bash
    $ python
   >>> import bobby.factor as bf
   >>> bf.get_prime_factor(11)
   >>> '11'
   >>> bf.get_prime_factor(222)
   >>> '2 3 37'
    ````
